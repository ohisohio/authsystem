To use this PHP Authentication System application

Create a folder in your localhost www folder (for wamp installations)

Name the folder snh

Clone this repository to that snh folder.

Go to localhost/snh/index.php

Register as a Patient or Medical Team

If you are an admin, visit localhost/snh//admin/index.php

Default username is admin, Default password is admin

Thank you for your time.

Only Medical Team in a Particular Department can see Patients who have booked in their own departments

Only Patients can pay bills
